## Image Prompt for Grok / Nano Banana

```
Create a split-screen comparison illustration representing Next.js vs WordPress for web development.

The image should feature:
- A clear visual divide between two worlds/sections
- LEFT SIDE (Next.js): Modern, sleek, minimalist design with React/Next.js logo, lightning bolt for speed, shield for security, clean code representation
- RIGHT SIDE (WordPress): More traditional CMS interface, WordPress logo, plugin icons, content management elements
- A scale/balance in the center suggesting decision-making
- Performance metrics and speed indicators favoring the modern side
- Code snippets abstractly integrated on both sides
- A subtle "VS" or comparison element in the center

Style: Tech comparison, balanced but suggesting modern superiority, suitable for developers and decision-makers.

Mood: Analytical, decision-oriented, modern vs traditional.

Color palette: Next.js side (black, white, clean blue accents), WordPress side (blue, gray, more colorful elements), neutral background.

Format: 16:9 horizontal, suitable for blog header/featured image.
```
